Moteur de Recherche et Recommandation de Séries

Description

Projet réalisé en groupe de 2

Conception et développement d'un moteur de recherche et système de recommandation de séries TV à partir de fichiers de sous-titres. Le projet inclut un important travail de NLP (TF-IDF, tokenization) pour structurer les données, une base de données PostgreSQL pour l'indexation, et une interface graphique Tkinter. Le tout a été déployé sur une machine virtuelle Linux.

Galerie du projet

Technologies Utilisées

  • Python (NLP, Tkinter)  
  • PostgreSQL  
  • TF-IDF pour le traitement de texte  
  • Linux (déploiement)  

Compétences Mobilisées

Gérer Conduire Collaborer

Fonctionnalités Clés

  • Recherche sémantique dans les sous-titres
  • Système de recommandation basé sur les préférences
  • Gestion de base de données performante
  • Déploiement sur une VM Linux
